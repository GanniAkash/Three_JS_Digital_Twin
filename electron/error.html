Error Loading Application

The application could not find the required files. This usually happens when the application was not built correctly.

Troubleshooting Steps:

  1. Make sure you've run npm run build before packaging the application
  2. Check that the dist folder exists and contains your built files
  3. Verify that the paths in your Electron configuration are correct
  4. Try running npm run electron:preview to test before packaging

For more information, check the console output (Ctrl+Shift+I or Cmd+Option+I to open DevTools).
